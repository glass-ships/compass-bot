### General Commands

???+ tldr "Slash Commands"
    | Command | Arguments | Description |
    | :--- | :--- | :--- |
    | `/avatar` | `None` | Get a user's Discord avatar |
    | `/banner` | `None` | Get a user's Discord banner |
    | `/ping` | `None` | Returns "Pong!" and the ping latency. |
